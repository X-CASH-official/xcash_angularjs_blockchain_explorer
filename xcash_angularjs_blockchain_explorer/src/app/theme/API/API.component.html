X-CASH Wallet Data API



Verify X-CASH Reserve Proof



Method: GET

URL: https://explorer.x-cash.org/verifyreserveproofapi?public_address=parameter1&reserve_proof=parameter2&data=parameter3

parameter1: The Public Address

parameter2: The Reserve Proof

parameter3: Any data that was used to create the reserve proof (leave empty if no data was used to create the reserve proof)
